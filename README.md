# Spaceship

Spaceship game in C, for Linux, based on @tsoj's, but with less code.

Use the arrows to move, try avoiding the asteroids.

## Compile

```sh
gcc main.c -o main
```

### Without cloning

```sh
wget https://raw.githubusercontent.com/donno2048/spaceship/master/main.c -O- | gcc -xc - -omain
```

## Run

```sh
./main
```
